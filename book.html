Optimal Event-Triggered Control Using Adaptive Dynamic Programming

Authors: Sarangapani Jagannathan, Vignesh Narayanan, Avimanyu Sahoo

Publisher: Taylor & Francis / Routledge

Year: 2024

This book presents a comprehensive treatment of optimal event-triggered control using adaptive dynamic programming (ADP). It covers theoretical foundations and practical applications of event-triggered control systems.

Table of Contents

  1. Introduction to Event-Triggered Control
  2. Fundamentals of Adaptive Dynamic Programming
  3. Event-Triggered Optimal Control for Linear Systems
  4. Event-Triggered Control for Nonlinear Systems
  5. Network-Based Event-Triggered Control
  6. Multi-Agent Systems with Event-Triggered Control
  7. Robustness and Stability Analysis
  8. Applications and Case Studies
  9. Future Directions and Open Problems
